# List of Examples

* `createdb.sql` —
  Initializes the database
* `sql_insert.js` —
  Inserts a random person
* `names.js` —
  Gives us random names
* `config.json` —
  Contains database configuration
* `sql_insert2.js` —
  Inserts two random persons in a transaction
* `sql_count.js` —
  Counts the persons
* `sql_list.js` —
  Lists all persons
* `sql_list2.js` —
  Lists all persons with an optional filter
